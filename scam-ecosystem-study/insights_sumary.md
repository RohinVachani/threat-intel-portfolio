# Key Patterns & Insights

- **Scaling Flywheel:** victim density -> low enforcement -> replicated scripts -> 
recruitment.
- **Division of Labour:** callers, closers, tech operators, mules reduce traceability.
- **Emotional Levers:** urgency, shame, and authority are primary attack surfaces.
- **OPSEC Decay:** repetition => normalization => mistakes => exposure.
- **Natural Selection & Reinvestment:** effective operators rise; money funds tech that 
increases efficiency.
